*Цель исправлений

Основная цель проведённых изменений — привести весь проект к:

единообразному стилю оформления кода (Google Java Style);

корректной структуре пакетов и классов;

строгому соответствию требованиям лабораторной работы;

повышению читаемости, надёжности и расширяемости кода;

добавлению Javadoc‑документации ко всем классам и методам;

устранению ошибок, дублирования логики и неоднозначных конструкций.

2. Общие улучшения по всему проекту

2.1. Приведение к Google Java Style

Были исправлены:

отступы (4 пробела → 2 пробела или наоборот, в зависимости от стандарта);

длина строк;

порядок импортов;

расположение полей, конструкторов и методов;

оформление фигурных скобок;

пустые строки и визуальные блоки.

2.2. Добавление Javadoc

Каждый класс получил:

краткое описание назначения;

Javadoc для всех публичных методов;

Javadoc для конструкторов;

корректные теги @param, @return, @throws (где нужно).
